Install Eliza Agent Framework

# Prerequisites: Node.js 23+, Git git clone https://github.com/elizaos/eliza.git cd eliza git checkout $(git describe --tags --abbrev=0) pnpm install cp .env.example .env

Configure Your Agent Character

Create characters/agent-trader.json with the $AGENT trading character.
